Drop your project images in this folder and name them as follows (recommended):

- `login_page.png` or `login_page.jpg`
- `dashboard.png`
- `daily_report.png`
- `architecture-diagram.png`
- `ips_iot.png`

Example usage in the repository `README.md`:

```markdown
![Login page](assets/images/login_page.png)
```

Tips:

- Keep images under ~1 MB each when possible.
- Prefer PNG for screenshots and JPG for photos.
- If an image contains sensitive info, don't add it to the repo.
